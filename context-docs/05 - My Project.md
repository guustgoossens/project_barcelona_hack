# 05 — My Project

## Name

**NeuralReach** (working title)

## Track

**Clay — "Signal over Noise"**

## One-Liner

The first outreach system powered by Meta's brain encoding model (TRIBE v2). Generates multiple email variants per candidate, runs them through a neural simulation to predict brain activation in curiosity and openness regions, and sends the version most likely to get a response.

## The Core Insight

Every other team will:
1. Enrich profiles from Clay's database
2. Ask Claude to write a "personalized" email
3. Send it

We do:
1. Enrich profiles from Clay's database → build personas
2. Ask Claude to write **3-5 variants** per candidate
3. Run each variant through **TRIBE v2** on a B200 GPU
4. Extract activation scores for emotion/curiosity/openness brain regions
5. Send the highest-scoring variant
6. Show the brain activation map as part of the outreach (the recipient sees their own neural score)

## What TRIBE v2 Actually Does Here

TRIBE v2 is Meta's open-source brain encoding model. It takes text input and predicts fMRI brain activity across 70,000 voxels. Specific brain regions are associated with:
- **Semantics** — does the message make sense to this type of person?
- **Emotions** — does it trigger an emotional response?
- **Curiosity** — does it activate novelty-seeking regions?
- **Social cognition** — does it activate theory-of-mind regions (crucial for outreach)?

We optimize emails to maximize activation in curiosity + social cognition regions, minimize threat/avoidance activation.

## Full Pipeline

```
Clay profile database
        ↓
Persona builder (Claude) — infer personality type, communication style, career stage
        ↓
Email variant generator (Claude) — 3-5 variants per candidate, different tones/hooks
        ↓
TRIBE v2 scoring (B200 GPU) — brain activation per variant
        ↓
Select optimal variant (highest curiosity + openness, lowest threat)
        ↓
Send via Clay outreach
        ↓
Lovable dashboard — show brain activation map + response tracking
```
